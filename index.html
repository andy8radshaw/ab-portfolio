// JOBR

Back

An online job search tool kit built utilising Django and React over 8 days. This was a group project, however, I was involved in every part of the app. The back-end was built using Django, Django Rest Framework and Python. The front-end was built using React Hooks throughout. I took charge of the Jobs, Contacts and Resources. As well as building the reusable Hooks components such as the modals for creation and deletion of items. For this project, we decided to use Semantic UI for the CSS Framework.

Deployed app: jobrrr.herokuapp.com

ReadMe: github.com/andy8radshaw/sei-project-4/blob/master/README.md

// Hikr

Back

A social networking and reviews site for hiking. This was a group project where we were challenged to build a full-stack application using the MERN stack in 8 days. Our group decided that we all wanted to be involved in the full-stack for this project. I took charge of the hikes for both front-end and back-end. The model was created based around CRUD features using MongoDB and Express. The frontend was built using React and Bulma/SASS for the styling. I made use of multiple third-party React packages including React.Moment, React-Mapbox-GL and React-Rating-Stars.

Deployed app: hikrrr.herokuapp.com

ReadMe: github.com/andy8radshaw/sei-group-project-3/blob/master/README.md

// Dinder

Back

A pair coded challenge to build an app within 48 hours. Using TheMealDB’s public API we built Dinder, an app that will help you decide what recipe to cook for your next meal, as a parody to the Tinder dating app. For this project, we used Figma to quickly build out a wireframe for the project, Insomnia to make sure we were receiving the correct responses from the API, and then React to build the frontend. Bulma/SASS was used for the styling. 

Deployed app: dinder.netlify.app/

ReadMe: github.com/andy8radshaw/project-2/blob/master/README.md
